# Customization Rules

- **Professional & Modern Standards**: For any future feature updates or modifications, ensure all UI components, state logic, animations, and interactions are built to a high professional and modern software standard. This includes:
  - **High-Fidelity Interaction**: Elements must be fully interactive (e.g., custom dragging, panning, zooming, and rotating for uploads) with smooth visual transitions.
  - **Premium UI Aesthetics**: Adhere strictly to premium web design principles including custom glassmorphism, responsive grid padding, and cohesive theme palettes.
  - **Clean Code & Robust Validation**: Build decoupled logic, perform strict validation (e.g., file sizing, input bounds), and write clean, warning-free React code.
